# MagicComputer

</div>


## 介绍

这是一个基于2026年春晚邓男子表演的手机计算器魔术的Android应用。魔术来自节目《惊喜定格》，其神奇之处在于，让所有跟着操作的观众，最终都在自己的手机上算出了同一个数字——2162227，而这串数字正好是节目播出时刻 2026年2月16日22点27分 的另一种写法。

## 功能

- 模拟魔术计算过程
- 引导用户进行计算步骤
- 显示最终结果

## 安装

1. 克隆此仓库：
   ```
   git clone https://github.com/yourusername/MagicComputer.git
   ```

2. 在Android Studio中打开项目。

3. 构建并运行应用。

## 使用

🎩 完整魔术流程
长按 ÷（进入魔术模式，无提示）

输入第一个数字（如1106）

按 +

输入第二个数字（如88396）

按 =（显示结果89502，历史记录显示）

按 +（显示"89502 +"）

随意输入一些数字（模拟乱点，如12345）

长按 ÷（自动替换为差值）

按 =（显示当前时间，如03150821）

最终效果：无论前两个数输入什么，最后都会得到当前的时间数字！
